# Chrome Web Store Listing Draft

## Extension name

Threej.in Games

## Summary

Browse and launch games from threej.in directly from your Chrome toolbar.

## Description

Threej.in Games gives you a quick launcher for the browser games available on `threej.in`.

Features:

- Fetches the live game catalog from the website
- Shows game covers and titles in a clean popup
- Includes search so users can find games quickly
- Opens the selected game directly in a new tab so play starts immediately

Single purpose:

- Help users browse and launch games hosted on `threej.in`

Permissions used:

- `https://threej.in/*`
  - Used only to fetch the live catalog and load cover images from the site

Privacy:

- The extension does not collect, store, or sell personal data
- Privacy policy text is in `PRIVACY_POLICY.md`
