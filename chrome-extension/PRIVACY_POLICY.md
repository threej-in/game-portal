# Privacy Policy for Threej.in Games

Last updated: 2026-04-20

## Overview

Threej.in Games is a Chrome extension that displays the game catalog from `https://threej.in` and opens selected games in a new browser tab.

## Data collection

This extension does not collect, store, or sell personal data.

## Data usage

The extension makes a read-only request to:

- `https://threej.in/api/games`

This request is used only to fetch the current game list, titles, images, and play links for display in the popup.

## Local storage

This extension does not currently store user data in Chrome storage, local storage, or any remote database.

## Third-party services

The extension loads catalog data and cover images from `https://threej.in`.
When a user clicks a game, the extension opens the selected game on `https://threej.in`.

## Permissions justification

- `host_permissions` for `https://threej.in/*`
  - Required so the extension can fetch the live game catalog and display game cover images from the website.
